# 🧾 Inventory Management System

---

## 🚀 Live Demo
👉 https://inventory-frontend-nine-alpha.vercel.app/

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios
- CSS / Tailwind (if used)

### Backend
- Node.js
- Express.js
- MongoDB (MongoDB Atlas)

- 📦 Product Management
- 🧾 Invoice Generation
- 💰 Mark Invoice as Paid
- ❌ Delete Paid Invoice
- 👤 User-specific data (secure)
